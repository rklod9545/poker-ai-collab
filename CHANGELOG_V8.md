# v8 变更说明（2026-04-23）

本轮重点：**多选板块** + **候选池** + **长期稳+近期爆专页** + **对错条 / 紧凑推导路径** + 诸多其他增强。

---

## 🎯 重点新功能

### 1. 多选板块（你点了就知道用了）
- 单选："一肖 / 一尾 / 一头"（只猜 1 个）
- **新增多选**：二肖 / 三肖 / 四肖 / 五肖 / 六肖 / 二尾 / 三尾 / 四尾 / 五尾 / 二段 / 三段 / 四段 / 二行 / 三行 / 二色
- 命中规则：**特码所属类别 ∈ 预测集合 → 命中**
- 真实测试：同一个基础公式，一肖 12% → 六肖 62%（随 N 线性递增）；五尾 54%

### 2. 🎒 候选池（侧边栏新菜单）
- 挖掘结果 → 勾选"加入候选池" → 去候选池再观察 → 满意的转正式库，不满意的删掉
- 批量挖掘器页面加了"🎒 加入候选池"按钮
- 候选池页支持：全选本页 / 转入正式库 / 删除 / 清空

### 3. 🔥 长期稳 + 近期爆（侧边栏新菜单）
- 专门挑"样本足 + 稳定性高 + 近20胜率 ≥ 全局胜率 × N 倍"的公式
- 8 个滑动条全可调：最小样本、最小稳定性、最小全局胜率、近期窗口（10/20/50）、爆发倍数（1.2-5×）、最大当前连黑、来源过滤、输出条数
- 真实测试：放宽阈值下筛出 17 条，TOP1 爆发倍数 2.28×

### 4. 📊 推导路径 + 近 15 期对错条
- **推导路径缩小到 12px** 小字号、灰底、默认折叠 — 之前太占屏
- **近 15 期对错条**：绿 ✓ / 红 ✗ 方块条直接画在公式详情、榜单卡片、单回测页走势图上方，旧→新排列，一眼看趋势
- 展示位置：公式库详情、正反榜卡片、单公式回测页、长期稳页的单条详情

## 🧬 底层增强

### 5. 生肖表按"生效起始期号"切换
- 之前生肖按自然年切换，农历初一前的几期会判错
- 新增 `core/zodiac_periods.py`，默认预置 2023/2024/2025/2026 四套 period
- 每套 period 带 `start_year / start_issue / end_year / end_issue / year_key`
- 回测、预测、统计全链路按 `(year, issue)` 查生效表
- 真实测试：`2026/5 → 2025 蛇年表`、`2026/30 → 2026 马年表`

### 6. 公式族 + 失效状态
- 自动为公式贴族标签：普通 / 函数 / 同期横向 / 跨期同列 / 对角线 / 十字 / 九宫格 / 特码锚点 / 聚合 / 交叉
- 失效状态四档：✓ 健康 / 👀 观察 / ⚠ 退化中 / ⛔ 暂停
- 观察：近20 略低于近100 5-15%，或当前连黑接近历史最大
- 退化中：近20 比近100 低 15% 以上
- 暂停：连黑 ≥ 15 且近20 比近100 低 20%+

### 7. 公式库批量操作（API 完成，UI 在下一轮做）
- `bulk_delete_formulas(ids)` / `bulk_update_favorite(ids, True/False)` / `export_formulas_csv(formulas)` 都可用
- CSV 导出带 UTF-8 BOM，Excel 打开不乱码

### 8. 预测聚类 + 号码频次统计
- 实战预测页底部新增两个区：
  - **共识聚类**：多条公式指向同一生肖/尾数时聚合，显示支持数 + 来源分布
  - **号码频次**：把所有预测（含 N 肖 / N 尾）展开成号码集合，配合历史近 500 期出现次数排序
- 真实测试：号码 02 被 14 条指向 + 历史 11 次，排第一
