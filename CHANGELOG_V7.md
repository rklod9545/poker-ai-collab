# v7 变更说明（2026-04-22）

本轮按用户明确需求做**针对性增强**，不做大范围重构。全部改动都有真实运行验证通过。

## ✅ 已实现

### 一、函数公式真正可嵌套
- `call_func` 节点可以作为任何子节点（另一个 `call_func` 的参数 / 二元三元的输入 / 包装块 / 映射块的输入）
- 公式构建器里的函数块，每个形参都可以从 `{常量/因子/已有块}` 里选，而"已有块"又可以是另一个函数块
- `describe()` / `fingerprint()` / `predict_next()` / `backtest()` 全部完整支持嵌套
- 真假公式校验**先展开 call_func 再检查**，嵌套藏 `lag=0` 也抓得到

验证样例：
```
F_sum2(
  ├─ F_absdiff(
  │   ├─ 上1期.平1
  │   └─ 上2期.平2
  │ )
  └─ 上1期.特码
)
```
→ 被正确判为真公式、回测 1089 样本、预测 2026/111→蛇、保存到库。

### 二、公式库 / 榜单 / 单回测都显示"当前预测下一期"
每条公式都显示：公式名 / 是否真公式 / 来源类型（普通/函数/交叉） / **预测目标期号** / **下一期预测结果** / 近50或近100胜率 / 当前连红连黑 / 综合评分 / 是否『已前进』（过期）。

### 三、历史更新后自动滚到下一期
- **预测目标期号从不写死在保存时**，始终基于当前 `history.csv` 最后一行动态计算
- `live_predict` / `formula_library` / `single_backtest` / 榜单 四个页面共用 `core/live_context.py` 里的同一个 `get_live_context(history)`，展示口径统一
- Streamlit session cache 的 key 以 `last_tail_key`（= "YYYY_NNN"）做后缀，**历史一更新缓存自动失效**
- 公式保存时会打上 `last_tail_key_when_saved`，之后若历史前进会显示 ⏳『已前进』徽标（保存时基于 2026/110）

**滚期测试实锤**：把 2026/111 加入 history 后，所有公式的预测期号从 `2026/111` 自动变 `2026/112`，预测结果也变了（蛇→龙）。

### 四、单公式回测页增强
- 顶部统一期号横幅
- 来源徽标（普通 / 函数 / 交叉）
- 函数公式：可展开查看**函数调用树**（含嵌套缩进）
- 交叉公式：可展开查看 3×7 格子图，**高亮用到的格子**
- 窗口评估改滑动条 10–500 + "使用全部"复选框
- 显示『已前进』标签

### 五、真假公式全链路校验
以下所有入口都跑 `is_predictive(expr)`（且**先展开 call_func**）：
- `formula_builder.py` 预览 + 保存前
- `storage.add_formula()` 自动 annotate
- `storage.load_formulas()` 懒加载时 annotate
- `single_backtest.py` 渲染公式概览时
- `auto_mine.py` / `batch_miner.py` 产出时（底层 lag 池就只有 1/2/3，不可能产假公式）
- `live_predict.py` 汇总前再跑一次兜底（拦截手改过的老数据）
- `rankings.py::evaluate_all` 榜单生成时

嵌套假公式拦截验证：`F_bad(x) = x + 特码(lag=0)`，调用时 `F_bad(上1期.平2)`，展开后 `lag=0` 暴露 → 被拦截，不会进榜。

### 六、正向榜 / 反向榜
位于 `📚 公式库 → 🏆 正向榜 / ⚠ 反向榜` tab：
- **正向榜**：只看真公式；可选按 综合 / 近100 / 近50 / 近20 降序；可按来源过滤；适合跟随
- **反向榜**：只看真公式；按**过热分**降序；过热分 = 0.30·(近20−近100) + 0.25·(当前连红/6) + 0.20·(1−稳定性) + 0.15·(最大连黑/20) + 0.10·近20
- 每条卡片都显示：名称 / 来源 / 真假 / 过期 / 6 格关键指标 / 下一期预测
- 支持点击展开查看推导路径 + 最近 10 期命中明细
- 榜长可选 3–20

反向榜真实产出示例（你的数据）：近20=45% / 近100=30% / 过热=0.459 的公式排第一。

### 七、交互/显示增强
- 每页顶部只显示一次期号横幅（不再重复贴）
- 横幅同时显示"基于最后一期：2026/110"和"当前预测下一期：2026/111"
- 每条公式都标注来源（函数/交叉的徽标颜色不同）
